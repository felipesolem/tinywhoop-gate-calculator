ℹ️ How to Use This Calculator ▼

📋 What This App Does

This calculator helps FPV TinyWhoop pilots determine exactly how much PVC tubing and connectors they need to build obstacle courses.

🎯 How to Use

  1. Enter Gate Size: Input the size per side in inches (19" is the standard TinyWhoop gate size)
  2. Select Obstacles: Choose how many of each obstacle type you want to build
  3. Calculate: Click the Calculate button to see your materials list
  4. Review Results: Get the total number of tubes, connectors, and PVC length needed

📏 Understanding Measurements

Full-size tubes: Equal to the gate size you enter (e.g., 19")

Floor tubes: Half of the gate size for florr stability (e.g., 9.5" for a 19" gate)

🔧 Connector Types

  • T-connectors (3 holes): Create T-shaped junctions
  • Elbow connectors (2 holes): 90-degree angle connections
  • 5-hole connectors: Cross/plus shape for complex structures
  • Cube connectors: Box corners for cube obstacles

⚙️ Gate Size

💡 19x19 inches is the standard size for TinyWhoop gates
